## Product Overview

参考瑞幸咖啡官网风格，打造"一起去也"机打鸡尾酒官方网站。采用现代简约设计，以酒红色为主题色，大留白强对比的视觉风格。网站无需收集客户信息，主要用于展示产品和技术优势，强化"品质统一，即刻出杯"的核心卖点，进行市场认知教育。

## Core Features

- **多页网站结构**：首页、产品介绍、技术优势、应用案例、联系我们/关于我们
- **首页展示**：核心卖点轮播图、产品概览、品牌形象展示
- **产品展示**：机打鸡尾酒系列产品展示，支持独立弹窗详细介绍
- **技术优势**：图文结合展示自动化、标准化等技术特点
- **应用案例**：实际应用场景和客户案例展示
- **联系方式**：展示电话、邮箱、地址、公众号和小红书二维码
- **SEO优化**：页面结构优化，关键词布局，提升搜索引擎排名

## Tech Stack

- **前端框架**：React + TypeScript
- **样式方案**：Tailwind CSS
- **UI组件库**：shadcn/ui
- **部署方式**：CloudBase 静态网站托管

## Tech Architecture

### System Architecture

采用前后端分离架构，纯静态页面部署。

- **表现层**：React组件化页面，响应式布局
- **静态资源层**：CloudBase 静态托管，CDN加速分发
- **SEO层**：Meta标签优化、语义化HTML、结构化数据

### Module Division

- **页面组件模块**：首页、产品、技术优势、案例、关于我们等页面组件
- **UI公共组件**：导航栏、页脚、轮播图、弹窗组件
- **内容数据模块**：产品信息、案例数据等静态内容管理

### Data Flow

用户访问页面 → React组件渲染 → 静态资源加载 → 用户交互（点击轮播/弹窗） → 页面状态更新 → 组件重新渲染

## Technical Considerations

### Performance Optimization

- **代码分割**：使用React.lazy和Suspense实现路由级代码分割
- **图片优化**：使用WebP格式，配置合适的图片尺寸
- **资源压缩**：启用Gzip压缩，减少资源体积
- **缓存策略**：配置浏览器缓存头，提升重复访问性能

### Security Measures

- **内容安全**：所有内容静态化，无动态数据提交，避免注入风险
- **HTTPS部署**：使用HTTPS协议，确保数据传输安全
- **资源防盗链**：配置Referer检查，防止资源被非法引用

### Scalability

- **静态架构**：纯静态页面易于扩展和维护
- **CDN加速**：支持高并发访问，无需担心服务器压力
- **内容管理**：通过修改JSON文件即可更新产品内容，无需代码改动

## Design Style

参考瑞幸咖啡官网的视觉风格，打造现代简约的"一起去也"品牌官网。整体采用大留白、强对比的排版方式，以酒红色(#B22222)作为主题色，营造专业、高端的品牌形象。

## Page Planning

1. **首页**：顶部导航栏、核心卖点轮播图、产品概览区、技术优势预览、品牌形象展示、页脚
2. **产品页**：产品列表展示、分类筛选、产品详情弹窗
3. **技术优势页**：技术特点图文展示、自动化流程图、标准化说明
4. **案例页**：案例卡片展示、应用场景分类、客户评价
5. **联系我们/关于我们**：联系方式展示（电话、邮箱、地址）、公众号和小红书二维码、品牌故事

## Page Block Design（以首页为例）

1. **顶部导航栏**：左侧Logo，右侧菜单（首页、产品、技术优势、案例、联系我们），支持吸顶效果
2. **核心轮播区**：大尺寸产品轮播图，展示核心产品形象和"品质统一，即刻出杯"标语，支持自动切换
3. **产品概览区**：4-6个产品卡片展示，点击跳转详情，悬停显示产品亮点
4. **技术优势预览**：3-4个核心优势图文展示，点击跳转技术优势页
5. **品牌形象区**：大尺寸品牌图片，强化品牌记忆
6. **页脚**：公司地址、联系方式、社交媒体二维码、版权信息

## Interaction & Animation

- **轮播图**：自动轮播（3秒间隔），支持手动切换，过渡动画平滑流畅
- **产品卡片**：悬停时卡片上浮并显示阴影，点击触发弹窗详情
- **页面滚动**：导航栏吸顶效果，页面元素滚动渐入动画
- **按钮交互**：悬停时颜色加深，点击时有轻微缩放反馈
- **弹窗动画**：淡入淡出效果，背景模糊处理
